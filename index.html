    Event Planner

    Een app om events aan te maken, op volgorde van datum weer te geven en te bewerken.

    Dit project bestaat uit een React page die data ophaalt en wegschrijft van en naar een JSON server. Om dit project voor demo te hosten, heb ik de React file ge-deployed via Netlify en de JSON server via Heroku. In een volgende versie zou ik onder andere user-authentication kunnen toevoegen via bijvoorbeeld Google Firebase.

    SuperPy

    ...is een in Python geschreven CLI gecontroleerde voorraad beheer app, waarbij producten per "batch" verwerkt kunnen worden en daarmee ook in volgorde van houdbaarheidsdatum kunnen worden verkocht. Per product kunnen verlies en winst worden weergegeven, onder andere via MathplotLib. Onderdeel van deze opdracht was ook het schrijven van een duidelijke manual.

    Momenteel leest en schrijft deze app alle data van en naar CSV files. In een volgende versie zou ik dit met een SQL database kunnen doen.
